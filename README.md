# LinkedIn Content Performance Analysis

## Overview

This project analyses one month of LinkedIn post data to identify
content and timing patterns that drive engagement and follower growth.

## Dataset

- One month of LinkedIn post-level data
- Daily follower growth metrics
- Data has been sourced from LinkedIn analytics exports

## Methodology

- Raw CSV data imported into PostgreSQL
- SQL used for exploratory and analytical queries
- Aggregated metrics used to drive insights

## Tools used

- PostgreSQL
- DBeaver
- VS Code
- SQL
